# 福建省新场景售彩系统 - 页面原型使用说明

## 项目概述

本原型系统基于福建省新场景售彩实施项目的需求文档、PRD、用户故事和系统设计方案设计，包含完整的页面原型，覆盖4类终端设备，共22个页面。

## 快速开始

1. 打开浏览器
2. 访问 `index.html`（导航索引页）
3. 点击对应卡片即可查看各终端页面原型

## 终端说明

### 1. 顾客点单终端（Android平板 - 10寸）
- **设计尺寸**: 适配1920×1200 (16:10) 平板横屏/竖屏
- **风格**: 简洁直观、大图展示、触控友好
- **主色调**: 体彩品牌色（红/蓝）+ 暖色调
- **页面数**: 6个

### 2. 收银台终端（Android/Windows - 15寸）
- **设计尺寸**: 适配1920×1080 (16:9) 收银台一体机
- **风格**: 专业高效、信息密集、操作便捷
- **主色调**: 商务蓝 + 白色背景
- **页面数**: 5个

### 3. 后台管理系统（Web端）
- **设计尺寸**: 响应式，适配1920×1080及以上
- **风格**: 数据可视化、专业管理后台
- **主色调**: 科技蓝 + 数据图表配色
- **页面数**: 9个

### 4. 移动端手机界面（直播手机）
- **设计尺寸**: 适配主流手机屏幕（375×812基准）
- **风格**: 简洁轻量、深色模式适配直播场景
- **主色调**: 深色模式 + 蓝色强调色
- **页面数**: 2个

## 交互说明

### 顾客点单终端
- 票种卡片可点击，支持分类筛选
- 购物车支持数量调整和删除
- 支付页包含倒计时和支付状态
- 直播页支持快捷加购
- 屏保页支持触摸唤醒

### 收银台终端
- 侧边栏导航可切换页面
- 订单列表支持状态筛选和操作
- 代客下单支持票种选择和支付方式切换
- 库存页支持入库和盘点操作
- 看板页数据实时展示

### 后台管理系统
- 左侧导航菜单分组展示
- 顶部面包屑导航
- 表格支持排序和分页
- 模态框支持新增/编辑/查看详情
- 图表支持时间维度切换

### 移动端手机界面
- 底部Tab导航切换页面
- 直播控制支持开始/结束
- 票种推荐支持多选
- 数据页支持时间筛选

## 技术实现

- **前端技术栈**: HTML5 + CSS3 + JavaScript
- **响应式布局**: Flexbox + Grid
- **样式框架**: 自定义CSS（模拟Ant Design风格）
- **图标**: Emoji / 自定义SVG
- **图表**: CSS实现（柱状图、饼图、环形图）
