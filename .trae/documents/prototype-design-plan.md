# 福建省新场景售彩系统页面原型设计计划

## 项目概述

基于福建省新场景售彩实施项目的需求文档、PRD、用户故事和系统设计方案，设计一套完整的页面原型系统，包含：
- 顾客点单终端（Pad设备）
- 收银台终端
- 后台管理系统（Web端）
- 移动端手机界面

## 二、设计规范

### 2.1 顾客点单终端（Pad - 10寸）
- **分辨率**: 1920×1200 (16:10)
- **设计尺寸**: 适配10寸平板横屏/竖屏
- **风格**: 简洁直观、大图展示、触控友好
- **主色调**: 体彩品牌色（红/蓝）+ 暖色调

### 2.2 收银台终端（15寸）
- **分辨率**: 1920×1080 (16:9)
- **设计尺寸**: 适配15寸收银台一体机
- **风格**: 专业高效、信息密集、操作便捷
- **主色调**: 商务蓝 + 白色背景

### 2.3 后台管理系统（Web端）
- **分辨率**: 响应式，适配1920×1080及以上
- **设计尺寸**: 桌面端优先
- **风格**: 数据可视化、专业管理后台
- **主色调**: 科技蓝 + 数据图表配色

### 2.4 移动端手机界面
- **分辨率**: 适配主流手机屏幕（375×812基准）
- **设计尺寸**: iPhone/Android通用
- **风格**: 简洁轻量、直播场景优化
- **主色调**: 深色模式适配直播场景

## 五、技术实现方案

### 前端技术栈
- **HTML5** + **CSS3** + **JavaScript**
- **响应式布局**: Flexbox + Grid
- **样式框架**: 自定义CSS（模拟Ant Design风格）
- **图标**: Font Awesome / 自定义SVG
- **图表**: ECharts（数据可视化页面）

### 设计原则
1. **一致性**: 相同功能组件保持视觉和交互一致
2. **可用性**: 界面清晰直观，操作流程顺畅
3. **响应式**: 适配不同设备尺寸
4. **美观性**: 符合现代UI设计趋势
5. **体彩品牌**: 融入体彩VI元素
